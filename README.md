# 📝 Todo App

A simple and clean Todo application built with **React**, featuring add, edit, delete, and mark-as-done functionality with **localStorage** persistence.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ☁️ Mark tasks as done (with strikethrough)
- 💾 Persists data in localStorage (survives page refresh)
- 🎯 Auto-focuses input on load

---

## 🛠️ Tech Stack

- React (Hooks: `useState`, `useEffect`, `useRef`)
- CSS (custom styling)
- React Icons (`react-icons`)
- localStorage API

---


## 🙌 Lessons Learned

- Controlled vs uncontrolled inputs in React
- Why `onClick={fn()}` vs `onClick={() => fn()}` matters
- How `useEffect` dependency array works
- Mutating state directly vs returning new objects in `map`
- How localStorage works with React state
